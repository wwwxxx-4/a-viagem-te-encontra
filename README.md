# A Viagem te Encontra

Plataforma de matching de viagens com algoritmo híbrido (swipe + quiz).

## Deploy

Este projeto é um site estático — basta fazer upload no Vercel.

## Stack

- HTML + CSS + JavaScript puro
- Zero dependências de framework
- 204 pacotes reais integrados
- Algoritmo de scoring: comportamento implícito (swipe) + intenção explícita (quiz)

## Estrutura

```
index.html    — Aplicação completa
vercel.json   — Configuração do Vercel
```
